Grandmaster Set Checklist

The Grandmaster Set Checklist is a browser-based tracking tool for the Magic: The Gathering - Avatar: The Last Airbender Universes Beyond release. It is designed as a print index and collection tracker, with one checklist entry for each distinct card treatment or variant.

What It Does

  • Lists the full set across multiple set codes, including playable cards and ancillary items.
  • Lets you search by set code, card number, card name, or treatment.
  • Supports per-column filtering so you can narrow the list quickly.
  • Supports sorting by set, number, name, treatment, and ownership state.
  • Lets you mark cards as owned with a checkbox.
  • Shows a live progress total for the cards currently visible in the table.
  • Includes a light and dark theme toggle.
  • Lets you save your checklist progress to a local .json file and load it again later.

How To Use It

  1. Open the Grandmaster Set Checklist.
  2. Use the search field to find a card by name, number, set code, or treatment.
  3. Use the column filter icons to narrow the table to a specific subset.
  4. Click the sort arrows if you want to reorder the table by a specific column.
  5. Tick the Owned checkbox for each card you have collected.
  6. Click Save to download your progress as a .json file.
  7. Click Load later if you want to restore that saved checklist.

Notes

  • The tool runs entirely in the browser and does not require an account.
  • Your checklist data is only stored in the file you choose to save.
  • The source for the tool is available on GitHub (external link).
